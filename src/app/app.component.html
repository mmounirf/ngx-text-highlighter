Ngx Text Highlighter

V1.0 Demo

Marker Style

Marker or text highlighter, can be displayed in two ways, as a fixed toolbar at the top of the text area, or floating above the selected text, like medium sharer toolbar. Use [markerStyle] input providing either 'fixed' or 'float'> as string

Marker Colors

By default text highlighter has 3 predefined colors to use for text highlighting (Red, Green and Blue), this can be extended by providing your own set of colors using [color] input. [color] accepts value of array of strings.
